# 方案 01：沉穩企業 SaaS 控制台

## 定位

適合 B2B SaaS、管理後台、CRM、ERP、權限管理、訂閱管理等需要高可信度與高可讀性的產品。

## 設計關鍵字

- 專業
- 穩定
- 清楚
- 可預期
- 高效率

## 視覺摘要

以淺色介面為主，搭配藍色作為主要行動色、灰階作為資訊骨架。整體不追求炫技，而是讓使用者快速完成工作。

## 適合頁面

- 儀表板首頁
- 使用者/團隊管理
- 權限設定
- 訂閱與帳務
- 稽核紀錄
- 通知中心

## 不適合

- 強品牌情緒的行銷頁
- 遊戲、娛樂、潮流風產品
- 需要大量沉浸式視覺效果的 AI 展示頁

## 實作優先順序

1. 先建立全域 layout：側邊欄、頂部列、內容容器。
2. 再建立資料元件：表格、篩選器、分頁、狀態標籤。
3. 最後補互動細節：dialog、toast、empty state、error state。
