# Pages：沉穩企業 SaaS 控制台

## Dashboard

結構：

1. Page header：標題、說明、主要 CTA。
2. KPI cards：3-4 張，包含數字、趨勢、輔助文字。
3. 主要圖表：營收/使用量/活躍度。
4. 次要列表：最近活動、待辦、告警。

## List Page

結構：

1. 標題列與新增按鈕。
2. 篩選器列：搜尋、狀態、日期、更多篩選。
3. Data table。
4. Pagination。
5. Empty / loading / error state。

## Detail Page

結構：

1. Breadcrumb。
2. Header：名稱、狀態、主要操作。
3. Tabs：概覽、設定、成員、紀錄。
4. 右側 metadata panel（可選）。

## Settings Page

- 使用左側 settings nav + 右側表單內容。
- 每個設定區塊用 card 分隔。
- 危險操作固定放在最下方 Danger Zone。
