# Pages：暗色 AI 工作台

## Chat Workspace

結構：

1. 左側 conversation list。
2. 中央 message stream。
3. 底部 prompt input。
4. 右側 context / settings panel。

## Prompt Playground

結構：

1. Header：模型、環境、儲存狀態。
2. 左欄 prompt editor。
3. 右欄 output preview。
4. 底部 logs / token usage。

## Agent Run Detail

結構：

1. Run summary：狀態、耗時、成本、模型。
2. Timeline：每個 step 的輸入輸出。
3. Artifacts：檔案、連結、diff。
4. Error panel：失敗時固定顯示。

## API Playground

結構：

1. Endpoint selector。
2. Request editor。
3. Response viewer。
4. Examples / docs side panel。
