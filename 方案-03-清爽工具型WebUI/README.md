# 方案 03：清爽工具型 WebUI

## 定位

適合線上小工具、轉換器、設定器、內容產生器、單一任務型 Web App。重點是低學習成本、快速完成、不要干擾。

## 設計關鍵字

- 極簡
- 直接
- 乾淨
- 快速
- 輕量

## 視覺摘要

大量留白、低裝飾、清楚輸入與結果區。使用者一進來就知道要輸入什麼、按哪裡、結果在哪裡。

## 適合頁面

- Converter / Generator
- 設定表單
- 單頁工具
- 文件格式化/解析
- 小型查詢工具

## 不適合

- 大型多角色後台
- 高品牌敘事 Landing
- 需要炫酷沉浸感的產品展示

## 實作優先順序

1. 建立單頁 layout：標題、說明、輸入區、操作列、輸出區。
2. 補齊表單 validation、copy/download/reset。
3. 加入歷史紀錄或範例，但不要搶主流程焦點。
