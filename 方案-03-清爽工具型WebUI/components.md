# Components：清爽工具型 WebUI

## Tool Card

包含：

1. 標題與一句說明。
2. Input 區。
3. Action bar。
4. Output 區。
5. Helper / tips。

## Input

- label 必須明確。
- placeholder 提供真實範例。
- textarea 預設高度 160px-240px。
- 支援貼上與清空。

## Output

- 結果區使用淡灰背景或白底灰框。
- 提供 copy button。
- 長文字要可捲動。
- 空狀態提示「輸入內容後會顯示結果」。

## Button

- Primary：執行、轉換、產生。
- Secondary：清空、重設、載入範例。
- Ghost：複製、下載、展開進階設定。

## Advanced Options

- 預設收合。
- 用 disclosure / accordion 呈現。
- 不影響主要流程。

## Toast

使用情境：

- 複製成功。
- 下載完成。
- 操作失敗但不需要阻塞流程。

## Example Pills

在輸入區下方提供 2-4 個範例按鈕，讓使用者快速開始。
