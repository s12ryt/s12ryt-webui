# Pages：清爽工具型 WebUI

## Single Tool Page

結構：

1. Header：產品名、簡短導航。
2. Intro：一句價值主張與簡短說明。
3. Tool card：輸入、操作、輸出。
4. Tips：使用提示或常見問題。
5. Footer：版本、隱私、回饋。

## Settings Tool

結構：

1. 左側或上方分類 tabs。
2. 表單區塊。
3. Sticky action bar：儲存/重設。
4. Unsaved changes warning。

## Result Page

結構：

1. Summary：結果摘要。
2. Result body：主要輸出。
3. Actions：copy、download、share。
4. Next steps：再次輸入、查看範例。
