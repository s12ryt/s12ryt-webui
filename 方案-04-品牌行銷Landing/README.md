# 方案 04：品牌行銷 Landing

## 定位

適合產品官網、服務介紹、等待名單、功能展示、價格頁、下載頁。目標是讓訪客快速理解價值並完成轉換。

## 設計關鍵字

- 品牌感
- 信任
- 敘事
- 轉換
- 記憶點

## 視覺摘要

使用強烈 hero、清楚 CTA、漸層或品牌圖形建立第一印象。頁面區塊要像故事：痛點、解法、功能、證明、價格、FAQ、CTA。

## 適合頁面

- 首頁 Landing
- Feature page
- Pricing page
- Changelog / Launch page
- Waitlist page

## 不適合

- 需要長時間操作的工作台
- 資料密度高的管理後台
- 純工具單頁

## 實作優先順序

1. 定義 hero 與主 CTA。
2. 建立區塊節奏：問題、解法、功能、信任、CTA。
3. 補社會證明、FAQ、footer。
