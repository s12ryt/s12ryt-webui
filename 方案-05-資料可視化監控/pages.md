# Pages：資料可視化監控

## Global Dashboard

結構：

1. Header：系統名稱、時間範圍、refresh。
2. KPI row。
3. Main chart area。
4. Alerts panel。
5. Recent events table。

## Service Detail

結構：

1. Service summary。
2. Uptime / latency / error charts。
3. Deployment or incident timeline。
4. Logs / drill-down table。

## Alerts Center

結構：

1. Severity filters。
2. Alert list。
3. Affected services.
4. Resolution actions.

## Report View

結構：

1. Report header。
2. Date range and export actions。
3. Charts and comparative tables。
4. Notes and anomalies。
