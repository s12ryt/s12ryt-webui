# Pages：教育學習課程平台

## Learning Dashboard

結構：

1. Welcome header：使用者與今日建議。
2. Continue learning card。
3. Course progress grid。
4. Upcoming deadlines。
5. Recommended courses。
6. Achievements / certificates。

規則：

- 最明顯 CTA 是繼續學習。
- 逾期提醒要清楚但不造成壓迫。
- 空狀態引導選課。

## Course Detail Page

結構：

1. Course hero：標題、講師、難度、時長、CTA。
2. Learning outcomes。
3. Curriculum。
4. Requirements。
5. Instructor profile。
6. Reviews / FAQ。
7. Sticky enroll/continue panel。

## Learning Player Page

結構：

1. Top bar：返回課程、進度、章節切換。
2. Main player/content。
3. Lesson metadata。
4. Notes / resources / discussion tabs。
5. Lesson navigation。
6. Curriculum side panel。

狀態：

- 未完成：顯示 mark complete。
- 已完成：顯示下一課。
- 鎖定：顯示解鎖條件。
- 載入失敗：提供重試。

## Quiz Page

結構：

1. Quiz header：題數、時間、進度。
2. Question content。
3. Answer options。
4. Explanation area（提交後）。
5. Navigation。
6. Submit confirmation。

規則：

- 未作答不可誤送，或送出前提醒。
- 結果頁顯示分數、解析、建議複習。

## Assignment Page

結構：

1. Assignment description。
2. Requirements checklist。
3. Upload / text editor。
4. Due date。
5. Submission history。
6. Feedback panel。

## Teacher Course Admin

結構：

1. Course metrics。
2. Student progress table。
3. Assignment review queue。
4. Quiz analytics。
5. Content management。

## Certificate Page

結構：

1. Completion statement。
2. Certificate preview。
3. Skills learned。
4. Share/download CTA。
5. Next course recommendation。
